---
title: Roadmap
description: Planned V1, V2, and V3 direction for branchbrief.
---

## V1

- Local CLI.
- Markdown output.
- JSON output.
- Git facts collection.
- Path and size based risk rules.
- Verification suggestions.
- GitHub Actions job summary and artifact workflows.
- Copilot-compatible context.
- Strong docs and examples.

## V2

- Optional LLM summaries.
- Safe data modes.
- Provider support for OpenAI, Anthropic, OpenRouter, and Ollama.
- PR comments.
- GitHub labels.
- Risk gates.
- `.github/copilot-instructions.md` generation.
- Config files.

## V3

- Policy-as-code.
- Multi-agent review flows.
- Semantic risk analysis.
- Baseline comparisons.
- Dashboards or static reports.
- Plugin rules.
- Monorepo support.
- Enterprise and privacy controls.
