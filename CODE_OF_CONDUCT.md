# Code of Conduct

## Standard

This project expects respectful, technical, good-faith participation.

Participants should:

- focus criticism on ideas, code, docs, and project direction
- be specific and actionable when giving feedback
- respect maintainer decisions about scope, safety, and release timing
- avoid harassment, threats, insults, or discriminatory language

## Enforcement

Maintainers may edit, hide, or remove comments and may restrict participation when behavior damages the project or its contributors.

For conduct concerns, contact the maintainer through the repository owner profile.
